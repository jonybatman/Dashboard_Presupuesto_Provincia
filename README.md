#Dashboard de Gestión de Organismos Descentralizados

Este proyecto es una aplicación web interactiva desarrollada con Dash y Plotly para la visualización de datos presupuestarios. Permite analizar de forma gráfica la ejecución de gastos (Corrientes, Capital y Financieros) de distintos organismos descentralizados.

#Características

Gráfico de Barras Horizontal: Comparativa de ejecución presupuestaria total por clasificación de gasto.
Gráfico Sunburst Dinámico: Desglose detallado de conceptos por organismo mediante un filtro interactivo.
Tabla de Datos: Visualización estructurada de los registros originales en formato de tabla corporativa.
Interfaz Dark Mode: Diseño moderno optimizado para la lectura de datos financieros.

#Tecnologías utilizadas

Dash - Framework de Python para aplicaciones analíticas.
Plotly Express - Biblioteca para gráficos interactivos.
Pandas - Manipulación y análisis de datos.
Dash DataTable - Componente para tablas interactivas.
